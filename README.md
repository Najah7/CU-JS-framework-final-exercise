# CU Final Project for JS framework

# How to run

## Local
1. clone the repo
1. create .env (you can refer to .env.example)
1. `npm install`
1. `npm run start`

## Docker
1. clone the repo
1. create .env (you can refer to .env.example)
1. `docker build -t node-server .`
1. `docker run -p 8080:8080 node-server`
